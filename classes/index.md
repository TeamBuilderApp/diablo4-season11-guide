## Contributing a Class Guide

Want to contribute a guide for a missing class or improve an existing one? Follow these steps:

1. **Read the [Contributing Guidelines](../CONTRIBUTING.md)** - Understand our standards and format requirements
2. **Use the [Template](template.md)** - Copy `classes/template.md` as your starting point
3. **Fill in Build Details** - Include all required sections with accurate information
4. **Mark Unknown Values** - Use `TO-VERIFY` for any values you're unsure about
5. **Cite Your Sources** - Reference Maxroll, Icy Veins, Wowhead, Fextralife, or AOEAH
6. **Submit a Pull Request** - Follow the branch naming convention `add/[classname]-guide`

## Frequently Asked Questions

### Q: Which class is best for beginners?
**A:** Barbarian and Necromancer are typically recommended for new players due to their straightforward mechanics and good survivability.

### Q: What does "TO-VERIFY" mean?
**A:** `TO-VERIFY` marks values that need verification from patch notes or community sources. See our [Contributing Guide](../CONTRIBUTING.md#to-verify-placeholders) for details.

### Q: Are these builds viable in Hardcore?
**A:** Most builds can be adapted for Hardcore by prioritizing defensive stats. Check the "Numeric Targets" section in each guide for defensive requirements.

### Q: How often are guides updated?
**A:** Guides are updated when major patches change class balance or when community members submit corrections and improvements.

### Q: Can I suggest a build variant?
**A:** Absolutely! Open a GitHub issue with the `enhancement` label or submit a pull request with your build variant.
